# AI Smart Parking

โปรเจคนี้เป็นระบบตรวจสอบสถานะช่องจอดรถด้วย Flask และ YOLOv8

## สิ่งที่มีใน repo นี้
- `app.py` - เว็บแอป Flask ที่รันเซิร์ฟเวอร์และสแกนภาพ
- `templates/index.html` - หน้าเว็บหลัก
- `static/` - ไฟล์สื่อประกอบ
- `zones.json` - เก็บพิกัดช่องจอดที่วาดไว้

## คำสั่งรันโปรเจค
1. ติดตั้งแพ็กเกจ
```bash
pip install -r requirements.txt
```
2. รันเว็บเซิร์ฟเวอร์
```bash
python app.py
```
3. เปิดเว็บเบราว์เซอร์ที่

http://127.0.0.1:5000

## หมายเหตุสำคัญ
- ไฟล์ `vdo.mp4` และ `vdo1.mp4` ถูกเว้นไว้จาก repo นี้เพราะขนาดเกิน limit ของ GitHub (100MB)
- หากต้องการทดสอบวิดีโอ ให้วางไฟล์ทั้งสองไว้ในโฟลเดอร์เดียวกับ `app.py`
- หากต้องการใช้กล้อง webcam ให้ตั้งค่าใน `MEDIA_PATHS` เป็น `0`
